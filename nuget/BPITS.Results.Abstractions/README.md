# BPITS.Results.Abstractions

Core abstractions and interfaces for BPITS.Results source generation.

## Installation

This package is automatically included when you install BPITS.Results:

```bash
dotnet add package BPITS.Results
```

You do not need to install this package directly.

## Documentation

See [BPITS.Results Documentation](https://github.com/BP-IT-Services/BPITS.Results/blob/main/nuget/docs/README.md) for complete usage information.

## Core Package

This package provides abstractions for [BPITS.Results](https://github.com/BP-IT-Services/BPITS.Results/blob/main/nuget/BPITS.Results/README.md).
